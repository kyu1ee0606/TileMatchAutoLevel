# 패턴 재생성 및 개별 검증 기능 개선

**날짜**: 2026-03-06
**버전**: v15.4

---

## 개요

프로덕션 대시보드에서 패턴 기반 레벨의 관리, 검증, 재생성 UI를 개선하였습니다.

### 주요 개선 사항
1. 패턴 정보 메타데이터 저장 및 재생성 시 보존
2. 메인 레벨 리스트에 개별 검증/재생성 버튼 추가
3. 재생성 모달 패턴 선택 UI 그리드 형태로 개선
4. 패턴 생성 모드 시 불필요한 대칭 옵션 숨김

---

## 변경 사항

### 1. 패턴 정보 저장 (`ProductionLevelMeta`)

레벨 메타데이터에 패턴 생성 정보를 저장하도록 타입을 확장했습니다.

**파일**: `frontend/src/types/production.ts`

```typescript
// 패턴 생성 정보
pattern_index?: number;                        // 사용된 패턴 인덱스 (undefined = 랜덤/패턴 없음)
pattern_type?: 'aesthetic' | 'geometric' | 'clustered';  // 패턴 타입
```

### 2. 패턴 정보 저장 로직

레벨 생성 시 사용된 패턴 정보를 메타데이터에 저장합니다.

**파일**: `frontend/src/components/ProductionDashboard/index.tsx`

- 초기 레벨 생성 시 `pattern_index`와 `pattern_type` 저장
- 재생성 시에도 패턴 정보 보존

### 3. 동일 패턴으로 재생성

순차 검증/재생성 시 기존 레벨에 패턴이 지정되어 있었다면 동일한 패턴으로 재생성합니다.

**로직 우선순위**:
1. 사용자 선택 패턴 (모달에서 선택한 경우)
2. 기존 레벨의 패턴 (`level.meta.pattern_index`)
3. 보스/특수 레벨 자동 선택 (10의 배수 레벨)

```typescript
// Pattern index: 사용자 선택 > 기존 레벨의 패턴 > 보스/특수 레벨 자동 선택
let patternIndex: number | undefined = userPatternIndex;
if (patternIndex === undefined) {
  // 기존 레벨에 패턴이 있으면 동일한 패턴으로 재생성
  if (level.meta.pattern_index !== undefined) {
    patternIndex = level.meta.pattern_index;
    patternType = level.meta.pattern_type || 'aesthetic';
  }
}
```

### 4. 미달 레벨 목록에 패턴 표시

"미달 레벨 재생성" 섹션의 레벨 목록에 패턴 정보를 표시합니다.

**테이블 헤더**:
| 선택 | 레벨 | 패턴 | 일치도 | 등급 | 목표 | 검증 | 상태 |
|------|------|------|--------|------|------|------|------|

**패턴 컬럼**:
- 패턴이 지정된 레벨: 패턴 아이콘 표시 (예: ❤️, ⭐, 🎄)
- 패턴 없는 레벨: `-` 표시
- 마우스 오버 시 패턴 이름 툴팁

### 5. 개별 검증 버튼

각 레벨에 개별 검증 버튼을 추가하여 선택한 레벨만 봇 시뮬레이션 검증을 실행할 수 있습니다.

**기능**:
- 클릭 시 해당 레벨만 봇 시뮬레이션 실행
- 검증 중에는 버튼이 회전 아이콘(⟳)으로 변경
- 검증 완료 시 결과 알림 표시
- 70% 이상: ✓ 통과 (success 알림)
- 70% 미만: ✗ 미달 (warning 알림)

**새로운 상태**:
```typescript
const [validatingLevels, setValidatingLevels] = useState<Set<number>>(new Set());
```

**새로운 함수**:
```typescript
const handleValidateSingleLevel = async (level: ProductionLevel) => {
  // 봇 시뮬레이션 실행 및 결과 저장
};
```

## UI 변경사항

### 메인 레벨 목록 (왼쪽 패널)

각 레벨 행에 다음 요소가 표시됩니다:

```
┌─────────────────────────────────────────────────────────────┐
│ 레벨 13                                                      │
│ 난이도: 0.134 (13%)                                          │
│                      [🎨] [검증] [재생성] [75%] [S] [생성됨]   │
│ [생성기]                                                      │
└─────────────────────────────────────────────────────────────┘
```

**버튼 설명**:
- **패턴 아이콘** (🎨/❤️/⭐): 패턴 생성된 레벨 표시, 툴팁에 패턴 이름
- **검증** (cyan): 봇 시뮬레이션으로 개별 레벨 검증
- **재생성** (orange): 해당 레벨만 재생성 (모달에서 패턴 선택 가능)
- **일치도** (75%): 봇 클리어율 일치도 (70%↑ 녹색, 50-70% 노랑, 50%↓ 빨강)
- **등급** (S/A/B/C/D): 난이도 등급
- **상태** (생성됨/테스트 대기): 현재 레벨 상태

## 사용 방법

### 패턴 기반 레벨 생성

1. 레벨 생성 시 "패턴 생성" 모드 선택
2. 원하는 패턴 선택 (하트, 별, 트리 등)
3. 생성된 레벨은 `pattern_index`가 저장됨

### 패턴 레벨 재생성

1. 순차 검증 또는 일괄 재생성 실행
2. 기존에 패턴으로 생성된 레벨은 자동으로 동일 패턴 사용
3. 수동 재생성 시 모달에서 패턴 변경 가능

### 개별 레벨 검증

1. "미달 레벨 목록"에서 원하는 레벨의 "검증" 버튼 클릭
2. 봇 시뮬레이션 실행 (설정된 반복 횟수만큼)
3. 결과 확인 (일치도 점수 및 통과 여부)

## 재생성 모달 UI 개선

### 모달 크기 확대

| 항목 | 기존 | 변경 |
|------|------|------|
| 모달 너비 | `max-w-md` (448px) | `max-w-2xl` (672px) |

### 패턴 선택 UI (드롭다운 → 스크롤 그리드)

**기존**: 드롭다운 클릭 → 팝업 열림 → 카테고리 탭 전환

**변경**: 스크롤 가능한 그리드가 모달 내에 직접 표시

```
┌─────────────────────────────────────────────────────────────┐
│ 🎲 자동 선택 (보스 레벨용 패턴 자동 지정)                      │
├─────────────────────────────────────────────────────────────┤
│ 기본 도형                                                    │
│ [▢] [◇] [⬭] [✚] [◎] [◈] [⌗] [⬡] [❤️] [⊤]                   │
│ 화살표                                                       │
│ [⬆️] [⬇️] [⬅️] [➡️] [⋀]                                       │
│ 별/천체                                                      │
│ [⭐] [✡️] [🌙] [☀️] [🌀]                                       │
│ ... (스크롤 가능)                                            │
└─────────────────────────────────────────────────────────────┘
```

**패턴 그리드 사양**:
| 항목 | 값 |
|------|-----|
| 높이 | `max-h-72` (288px) |
| 열 수 | 10열 |
| 아이콘 크기 | `text-xl` + `p-2` |
| 선택 효과 | 파란색 배경 + 링 + `scale-110` |
| 호버 효과 | `scale-105` 확대 |

### 대칭 모드 조건부 표시

| 생성 모드 | 대칭 옵션 | 이유 |
|-----------|----------|------|
| 빠른 생성 | 표시 | 레이어별 다른 패턴이라 대칭 의미 있음 |
| 패턴 생성 | 숨김 | 패턴 자체가 형태를 결정함 |
